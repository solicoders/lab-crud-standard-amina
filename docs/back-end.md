# Back-end

## Lignes de commande utilisées


- Création de la table "Module"

```bash
php artisan make:model Module -m
php artisan migrate

```


- Création du modèle "Module"

```bash
php artisan make:model Module -m
```

- Controller "Module"

```bash
php artisan make:controller ModulesController
```
